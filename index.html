Privacy Policy for DuckIt

Effective Date: January 29, 2026

DuckIt respects your privacy. This extension is designed to work entirely on your local device and does not collect, store, transmit, or share any personal data.

1 Data Collection

DuckIt does not collect any of the following:

  • Personally identifiable information (name, email, address, etc.)
  • Browsing history
  • Audio content
  • Website content
  • User activity such as clicks, keystrokes, or navigation
  • Location information
  • Financial or authentication information

All processing is done locally within your browser.

2 Local Storage

DuckIt uses Chrome’s local storage only to save:

  • The selected Music Tab
  • User preferences (ducking mode, volume level, enable/disable state)
  • Temporary session state

This data never leaves your device and is never transmitted to any server.

3 Data Sharing

DuckIt does not:

  • Sell user data
  • Transfer user data to third parties
  • Share data with advertisers
  • Use analytics or tracking services

There is no external communication of any kind.

4 Remote Code

DuckIt does not use remote code.

All JavaScript and functionality are bundled directly inside the extension package.

5 Permissions Usage

DuckIt requests browser permissions only to:

  • Detect audio activity in tabs
  • Control audio behavior (mute, pause, duck, restore)
  • Store user preferences locally
  • Show notifications for user awareness

These permissions are strictly limited to the extension’s single purpose of intelligent audio management.

6 Security

Because DuckIt does not transmit or store personal data externally, there is no risk of data leakage or tracking.

The extension operates completely offline after installation.

7 Changes to this Policy

If this privacy policy changes in the future, the updated version will be published at the same URL.
